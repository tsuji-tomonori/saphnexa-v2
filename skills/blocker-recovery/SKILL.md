# Blocker Recovery

## Purpose
Treat blockers as blockers, not as completion.

## Recovery ladder
1. Re-read request and checklist.
2. Inspect repository docs/scripts for setup and conventions.
3. Try safe, non-destructive workaround.
4. Narrow failing command for diagnosis.
5. Continue independent unblocked tasks when safe.

## Blocked report
- Blocked item
- Cause
- Evidence
- Attempted workaround
- Remaining work
- User action needed
