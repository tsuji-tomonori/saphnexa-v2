# なぜなぜ分析 Skill Pack

研究・実務ガイドラインを踏まえて、なぜなぜ分析を「5回なぜを聞く」だけで終わらせず、事実確認、分岐因果、システム要因、強い再発防止策まで進めるためのスキルパックです。

## 構成

- `SKILL.md`：AIエージェントまたはファシリテーター向けの実行スキル
- `research_brief.md`：調査した研究・ガイドラインと設計判断
- `templates/worksheet.md`：分析ワークシート
- `templates/report_template.md`：報告書テンプレート
- `templates/facilitation_prompts.md`：会議で使う問い・プロンプト集

## 想定用途

品質不良、業務ミス、プロジェクト遅延、IT障害、安全インシデントなど、発生済み問題の再発防止に使います。重大事故、法務・規制リスク、複雑な社会技術システムの事故では、単独のなぜなぜ分析で結論を出さず、RCA、FTA、FMEA、WBA、HFACS、人間信頼性工学などの拡張分析と併用してください。
